Para cadastrar um novo dispositivo:

    - Cadastrar um novo projeto no Blynk, criar os campos de usuário, deviceId, descrição e flowRate
    - Gerar o token de autenticação no Blynk
    - No código para ser compilado no Arduino, colocar os valores para o usuário, deviceId, descrição e token de autenticação com seus respectivos pinNumbers
    - Compilar o código no Arduino
    - Instalar o arduino em uma tubulação
